# Polyfills

eo-locale based on features of Intl object.
[Check](https://caniuse.com/#feat=internationalization) that your browser supports Internationalization API.

If no, add polyfills.

## Install

### With npm

```sh
npm i @eo-locale/polyfills --save
```

### Or with yarn

```sh
yarn add @eo-locale/polyfills
```

## Usage

### Server side

```js
import { serverPolyfill } from '@eo-locale/polyfills';

serverPolyfill(['en', 'ru']);
```

### Client side

```js
import { clientPolyfill } from '@eo-locale/polyfills';

clientPolyfill().then(() => {
  // Init your application
});
```
