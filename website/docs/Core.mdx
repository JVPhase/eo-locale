---

You can use **@eo-locale/core** package without any dependencies.
Just create instance of class `Translator`, and import wherever you need.

```ts
import { Translator } from '@eo-locale/core';

export const translator = new Translator('en');

translator.formatNumber(1000);
translator.formatDate(new Date());
translator.translate('hello', {
  defaultMessage: 'Hello World!',
});
```
