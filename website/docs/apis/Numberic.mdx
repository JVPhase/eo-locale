`Numeric` component provides ability to format numeric values.
Full list of supported properties you can find on [MDN](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/NumberFormat) page

## React / Preact

```jsx
import { Numeric } from '@eo-locale/react';

export function SomeComponent() {
  return <Numeric language="en" value={123456789.321} />;
}
```

## React Native

```jsx
import { Numeric } from '@eo-locale/react-native';

export function SomeComponent() {
  return <Numeric language="en" value={123456789.321} />;
}
```

## Result

Component will display <LanguageSelector><Highlight><Preview/></Highlight></LanguageSelector>

export const Preview = () => {
  const language = useLanguage();
  return <Numeric language={language} value={123456789.321} />
};
