Sometimes you need ability to render some strings not as components.
For example, placeholder of input. In this cases `useTranslator` might be very helpful.
That hook returns instance of class [`Translator`](../core/translator).

## React / Preact

```jsx
import { useTranslator } from '@eo-locale/react';

export function SomeComponent() {
  const translator = useTranslator();

  return <input placeholder={translator.formatNumber(1000)} />;
}
```

## React Native

```jsx
import { useTranslator } from '@eo-locale/react-native';

export function SomeComponent() {
  const translator = useTranslator();

  return <input placeholder={translator.formatNumber(1000)} />;
}
```
