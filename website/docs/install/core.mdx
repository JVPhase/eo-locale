title: Core Package
---

**`eo-locale`** has a core package contains functions that can help you translate words easily wherever you need.
(React, Preact, React Native, even Valina JS) You can install it with NPM or Yarn.

### with NPM

```sh
npm i @eo-locale/core --save
```

### with Yarn

```sh
yarn add @eo-locale/core
```
