title: Quick start
---

Add provider in root of your application

```jsx
import { Text, TranslationsProvider } from '@eo-locale/react';

const locales = [
  {
    language: 'en',
    messages: {
      hello: 'Hello {name}!'
    }
  },
];

<TranslationsProvider language="en" locales={locales}>
  <span>
    <Text id="hello" name="World" /> // Helo World!
  </span>
</TranslationsProvider>
```
